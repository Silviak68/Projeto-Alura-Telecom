# Projeto-Alura-Telecom
Projeto da trilha de ETL do programa Ciência de Dados oferecido pelo programa One da Alura em conjunto com a Oracle
O objetivo do projeto é descobrir por que há uma grande evasão de clientes, acima de 25%, de uma empresa de telecom. 
Os dados analisados vieram de um dataset, que tem dados antigos, observação que foi importante para nossa análise uma vez que essa caracaterística explica a preferência por serviços de internet via DSL (linha telefônica) em detrimento de fibra óptica.
Observamos que há vários fatores que contribuem para a elevada taxa de evasão, como a possibilidade de compartilhar o serviço com outros membros da família e o método de pagamento. 
Também verificamos que o primeiro mês do contrato é crucial para a fidelização dos clientes pois mais da metade dos contratos novos são suspensos nesse período como mostra o gráfico abaixo. O primeiro trimestre é o que mais influi na fidelização, ou não, da clientela.
